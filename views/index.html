Cómo acceder desde un sitio externo

Utiliza este código para hacer una solicitud desde cualquier sitio:

fetch('{{publicUrl}}/outbound-call', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: "Eres un agente de ventas llamando para ofrecer un producto nuevo",
    first_message: "Buenos días, mi nombre es Valeria Rodríguez, soy representante comercial de Smoke Off, me comunico con {{user_name}}?"
  })
})
.then(response => response.json())
.then(data => console.log('Respuesta:', data))
.catch(error => console.error('Error:', error));
